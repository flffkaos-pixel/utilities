대소문자 변환기Case Converter

텍스트를 다양한 케이스로 변환하세요. 실시간 변환, 클릭 한 번으로 복사.Convert text between different cases. Real-time conversion, copy with one click.

info 케이스 종류 설명Case Types

UPPERCASE: 모든 문자를 대문자로All letters uppercase
lowercase: 모든 문자를 소문자로All letters lowercase
Title Case: 각 단어의 첫 글자만 대문자Capitalize first letter of each word
Sentence case: 첫 문장만 대문자로Capitalize first sentence only
camelCase: 단어를 붙여 쓰고 각 단어의 첫 글자를 대문자로 (첫 단어 제외)Words joined, each word capitalized except first
PascalCase: camelCase와 같지만 첫 단어도 대문자Like camelCase but first word capitalized
snake_case: 단어 사이를 밑줄(_)로 연결Words separated by underscores
kebab-case: 단어 사이를 하이픈(-)으로 연결Words separated by hyphens
