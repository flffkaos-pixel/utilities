Frequently Asked Questions

무료 유틸리티는 무엇인가요?

What is Free Utilities?

Free Utilities는 개발자와 직장인을 위한 100개 이상의 무료 온라인 유틸리티 도구 모음입니다. 텍스트 변환, QR 코드 생성, 비밀번호 생성, JSON 포맷터, 단위 변환, 계산기 등 다양한 도구를 설치 없이 브라우저에서 바로 사용할 수 있습니다.

Free Utilities is a collection of 470+ free online utility tools for developers and office workers. Tools include text converters, QR code generators, password generators, JSON formatters, unit converters, calculators, and more. All tools run in your browser with no installation required.

설치가 필요한가요?

Do I need to install anything?

설치가 필요 없습니다. 모든 도구는 웹 브라우저에서 직접 작동하며, 데스크톱, 태블릿, 모바일 등 모든 기기에서 사용할 수 있습니다.

No installation required. All tools work directly in your web browser on any device - desktop, tablet, or mobile.

데이터는 안전한가요?

Is my data safe?

네, 모든 데이터 처리는 사용자의 브라우저 내에서만 이루어집니다. 어떤 데이터도 서버에 전송되거나 저장되지 않으므로 안전하게 사용할 수 있습니다.

Yes. All data processing happens locally in your browser. No data is ever sent to or stored on our servers. Your privacy is our top priority.

회원가입이 필요한가요?

Do I need to register?

회원가입이 전혀 필요 없습니다. 모든 도구를 로그인 없이 즉시 무료로 사용할 수 있습니다.

No registration required. All tools are free to use immediately without signing up or logging in.

텍스트 도구(대소문자 변환, 글자 수 세기), 개발자 도구(JSON 포맷터, Base64 인코더, QR 코드 생성), 계산기(BMI, D-Day, 할인율), 단위 변환기, 비밀번호 생성기 등 100개 이상의 다양한 도구를 제공합니다.
