HTML 인코딩이 왜 필요한가요?Why is HTML encoding needed?
사용자 입력을 웹페이지에 표시할 때 XSS(교차 사이트 스크립팅) 공격을 방지하기 위해 HTML 인코딩이 필요합니다. <, >, & 등의 문자를 안전한 엔티티로 변환합니다.HTML encoding prevents XSS (Cross-Site Scripting) attacks when displaying user input on web pages. It converts <, >, & into safe entities.
