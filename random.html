🎲 랜덤 번호란 무엇인가?

랜덤 번호는 예측 불가능한 순서로 생성되는 숫자입니다. 도박, 게임, 통계 샘플링, 보안 등 다양한领域에서 활용됩니다. 진정한 무작위성(True Random)은 물리적 현상(열잡음, 방사성衰変 등)에서 비롯되며, 컴퓨터에서 생성되는 의사 랜덤(Pseudo-random)은 알고리즘에 기반합니다.

💻 컴퓨터의 랜덤 생성

컴퓨터의 랜덤 번호 생성은 크게 두 가지로 나뉩니다. 진짜 랜덤 번호 생성기(TRNG)는 물리적 현상을 기반으로 Truly 예측 불가능한 숫자를 생성합니다. 의사 랜덤 번호 생성기(PRNG)는 시드(seed) 값으로 시작하여 알고리즘을 통해 다음 값을 예측합니다. 대부분의 응용 프로그램에서는 PRNG로 충분합니다.

JavaScript의 Math.random()은 0부터 1 사이의 부동소수점을 반환하며, 이를 원하는 범위로 변환하여 사용합니다. 보안이 중요한 경우에는 crypto.getRandomValues()를 사용해야 합니다.

🎲 What are Random Numbers?

Random numbers are generated in unpredictable sequences. Used in gambling, gaming, statistical sampling, security, and more. True randomness comes from physical phenomena (thermal noise, radioactive decay), while computer-generated pseudo-randomness is algorithm-based.

💻 Computer Random Generation

Computer random number generation falls into two categories. True Random Number Generators (TRNG) use physical phenomena for genuinely unpredictable numbers. Pseudo-Random Number Generators (PRNG) start with a seed and predict next values via algorithm. PRNG suffices for most applications.

JavaScript's Math.random() returns floating points between 0 and 1—transform to desired range. For security-critical uses, use crypto.getRandomValues().
