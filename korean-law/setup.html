채팅 구독 API 설정

서버 설정 가이드

대한법률AI가 정상 작동하려면 백엔드 Worker에 API 키를 설정해야 합니다.

key 1. OpenAI API 키 발급

AI가 법률 질문을 이해하고 답변을 생성하는 데 필요합니다.

  1. platform.openai.com에 접속
  2. 회원가입 후 API 키 생성 (Secret Key)
  3. 생성된 키를 복사 (sk-... 형태)
⚠️ OpenAI API는 사용량 기반 과금입니다. 일반적으로 법률 질문 한 건당 약 10~50원 정도 소요됩니다.

gavel 2. 법제처 Open API 인증키 발급

대한민국 법령과 판례를 검색하는 데 필요합니다 (무료).

  1. 법제처 Open API 접속
  2. 회원가입 후 로그인
  3. "Open API 사용 신청" 버튼 클릭
  4. 신청서 작성 → 인증키(OC) 발급 (예: honggildong)

cloud_upload 3. Cloudflare Worker 배포

백엔드를 Cloudflare Workers에 배포합니다.

방법 A: Cloudflare Pages에 함께 배포

이미 Cloudflare Pages를 사용 중이라면 _worker.js 파일이 Pages와 함께 배포됩니다.

  1. Cloudflare Dashboard → Workers & Pages
  2. 환경변수 설정:
    • OPENAI_API_KEY = sk-...
    • LAW_OC = your-law-oc-key

방법 B: 별도 Worker 배포

# wrangler.toml
name = "korean-law-api"
main = "_worker.js"
compatibility_date = "2025-01-01"

[vars]
OPENAI_API_KEY = "sk-..."
LAW_OC = "your-law-oc-key"

그리고 프론트엔드 index.html에서 fetch 주소를 Worker URL로 변경하세요.

빠른 시작

API 키 없이도 다음과 같은 기능을 이용할 수 있습니다:

  • ✅ 하루 1회 무료 질문 (localStorage 기반 체험)
  • ✅ 채팅 UI 미리보기
  • ✅ 구독 페이지 확인

실제 법률 검색이 필요하시면 위 설정을 완료해주세요.
