안전한 비밀번호 만드는 방법

비밀번호 생성기에서 12자 이상, 숫자+특수문자+대소문자를 포함하도록 설정하세요. 매번 다른 비밀번호를 만들고, 모든 계정에 같은 비밀번호를 쓰지 마세요. 나중에 관리하기麻烦하지만安全합니다.

Set the password generator to 12+ characters with numbers, special chars, and upper/lower case. Use a different password for each account. It's more work but much safer.

이미지 크기 줄여서 웹사이트 빠르게 만들기

웹페이지가 느리면 이미지를 확인해보세요. 이미지 압축기에서 품질을 70-80%로 낮추면 크기가 50% 이상 줄어듭니다. 화질은 거의 차이가 없고 网页속도는 빨라져요.

If your website is slow, check the images. Use the image compressor at 70-80% quality to reduce size by 50%+. Quality stays nearly the same but pages load much faster.
